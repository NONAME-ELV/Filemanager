# E.L.V FILEMANAGER v12.1
> *Cyberpunk Edition*

E.L.V Filemanager v12.1 adalah tool manajemen file berperforma tinggi yang dirancang untuk efisiensi operasional dan estetika *cyberpunk*. Dibangun dengan fondasi `ELV_ENGINE`, proyek ini memberikan kecepatan dan stabilitas bagi pengguna yang membutuhkan manajemen data yang taktis.

## ЁЯЪА Cara Penggunaan

1.  Upload file `cyberpunk.php` ke direktori target.
2.  Akses file melalui *browser* Anda.
3.  Gunakan *action toolbar* untuk melakukan manipulasi file:
    * **UNZIP**: Ekstraksi arsip dengan cepat.
    * **WIPE**: Penghapusan data taktis (*Tactical Deletion*).
    * **DOWNLOAD**: Pengunduhan data yang aman.
